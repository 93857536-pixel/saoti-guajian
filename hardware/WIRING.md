# 扫题挂件接线说明

固件引脚以 [`include/pins.h`](include/pins.h) 为准。接线前**断电**。

## 1. 屏幕 ST7789（微雪 1.3" / 同类 SPI）

| 屏幕 | ESP32 丝印 |
|------|------------|
| VCC | 3V3 |
| GND | GND |
| DIN / MOSI | **1** |
| CLK / SCLK | **3** |
| CS | **38** |
| DC | **39** |
| RST | **40** |
| BL | **41** |

勿占用 GPIO 35–37（OPI PSRAM）。

## 2. 摄像头 OV5640（DVP）

供电：**3.3V + GND**（禁止 5V）。

| ESP 功能 | GPIO | 微雪 Camera Board (C) 丝印 | 正点原子常见丝印 |
|----------|------|---------------------------|------------------|
| SIOD | 4 | SIOD | SDA / SIOD |
| SIOC | 5 | SIOC | SCL / SIOC |
| D0 | 6 | **D2** | D0 |
| D1 | 7 | **D3** | D1 |
| D2 | 8 | **D4** | D2 |
| D3 | 9 | **D5** | D3 |
| D4 | 10 | **D6** | D4 |
| D5 | 11 | **D7** | D5 |
| D6 | 12 | **D8** | D6 |
| D7 | 13 | **D9** | D7 |
| XCLK | 15 | XCLK | XCLK |
| VSYNC | 16 | VSYNC | VSYNC |
| HREF | 17 | HREF | HREF |
| PCLK | 18 | PCLK | PCLK |
| RST / PWDN | — | 可悬空 | 可悬空 |

微雪 C 型：约 35.7×23.9 mm，镜头两侧闪光灯，双排 2×9 针；数据脚是 **D2–D9**，不是 D0–D7。  
补光灯由模组内 **OV5640 STROBE** 驱动（无需另接 GPIO）；固件拍照前测光偏暗时自动点亮。  
自动对焦：微雪 C 型需保证模组 **AF-VCC（约第 24 脚）接 3.3V**，固件会加载 AF 固件并在拍照前单次对焦。

## 3. 4G 模块 FS-MCore-A7670G（全球版 AT）

| 模块丝印 | ESP32 | 说明 |
|----------|-------|------|
| VIN | **独立 5V ≥2A** | 不要只靠开发板 USB 硬撑 |
| GND | GND | 与 ESP **共地** |
| RX | **21** | ← ESP TX（交叉） |
| TX | **2** | → ESP RX（交叉） |
| PEN | **48**（或直接 3V3） | **使能，必须常高**；固件上电拉高 GPIO48 |
| PWK | **47** | 开机键；固件拉低约 1.2s |
| NET | **悬空** | 不要接到 GPIO2（会和 UART RX 冲突） |

注意：

- TX/RX **交叉**；不要接到板子标 USB 的 `TX`/`RX`（那是 USB 口）。
- **旧店家表作废**：`模TX→47`、`PWK→41`、`NET→2` 会与当前屏/串口冲突，勿再按那个接。
- 蓝灯亮才算模块开机；只有红灯通常还没起来。

### 3.1 先用 FreeAT 验证模块本身（强烈建议）

商家工具 `FreeAT_2.3.0.exe` 是 **Windows** 串口 AT 助手（Mac 不能直接双击跑）。

1. Windows 电脑安装模块 USB 驱动（店家资料包）。
2. 模块 **Type-C USB** 插电脑；**VIN 仍建议另接 5V≥2A**，GND 共地。
3. **PEN 接 3.3V**（或接 ESP 的 3V3），必要时手动短接 **PWK** 到 GND 约 1～2 秒开机，等蓝灯。
4. 打开 FreeAT → 选 COM 口 → **115200 8N1** → 发送 `AT`，应回 `OK`。

| FreeAT 结果 | 含义 | 下一步 |
|-------------|------|--------|
| USB 有 `OK` | 模块本身正常 | 再按上表接 ESP UART |
| USB 也无响应 | 供电/开机/驱动问题 | 先别查 ESP 固件 |

Mac 上等价做法：烧录 `esp32s3-modem-bridge`，用串口监视器发 `AT`。

## 4. 电池电量（分压 → GPIO14）

开发板**不能**直接测电池电压，需两只电阻分压后接 ADC：

```text
电池正极 B+ ── 100kΩ ──┬── GPIO14（ESP 丝印 14）
                        │
                       100kΩ
                        │
                       GND
```

| 参数 | 值 |
|------|-----|
| 分压比 | 2（固件 `BAT_DIVIDER_RATIO`） |
| 空电 / 满电 | 3.30V / 4.20V（锂电） |
| 显示 | 屏左上角 `85%`；未接线显示 `--%` |

注意：电阻接到**电池端（升压前）**，不要接到 5V 升压后。两电阻建议 ≥100k，减小耗电。

**为何用 ADC2（GPIO14）：** ESP32-S3 的 ADC1（GPIO1–10）已被 LCD/摄像头占满，只能用 ADC2。WiFi SoftAP（答案页热点）开启时 ADC2 会受射频干扰，固件会**冻结电量采样**（保持上次读数），热点关闭后再更新。若以后改板，优先把电池分压迁到空闲的 ADC1 脚。

## 5. 按键（2 位独立按键模块）

实物：[天猫 2 位独立按键模块](https://detail.tmall.com/item.htm?id=610117189167)（电子积木双大方键板，**不是**单颗 6×6）。

常见排针：`GND` / `KEY1` / `KEY2`（按下拉低；ESP `INPUT_PULLUP`）。若板子有 `VCC`，接 **3V3**。

| 模块 | ESP | 功能 |
|------|-----|------|
| KEY1 | **GPIO0**（丝印 0 / BOOT） | 短按扫题；长按≈3s 测 AI |
| KEY2 | **GPIO42** | 短按固定题图测 AI |
| GND | GND | 共地 |
| （可选）板载 BOOT | — | 与 KEY1 同脚，后壳腰圆孔备用 |

### 3D 打印外壳要点（v7.5）

- 后壳**左侧壁**：双键方窗 + 内侧整板凹座（口袋约 42×24，键距 15.5）。
- ESP USB 仍出**右侧壁**；板载 BOOT 腰圆孔保留。
- 上电瞬间勿按住 GPIO0（会进下载模式）。

电源开关仍用电池正极拨动开关，与拍照键分开。

可打印文件：[`case/PRINT.md`](case/PRINT.md)、[`case/COMPONENTS.md`](case/COMPONENTS.md)。
