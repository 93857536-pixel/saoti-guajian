★★★ 最重要：禁止把 4G 板直接叠插在 ESP32 排针上 ★★★
================================================================================

  · 必须用杜邦线/飞线逐根连接，不能整板对插！
  · 对插容易错一格，会出现 GND 接到 VIN，烧模块！
  · 4G 的 VIN 必须接 5V（右侧 5V 脚），不能靠叠插取电
  · 4G 的 TX/RX 必须交叉接到 47/21，不能接到左侧 TX/RX 字样（那是 USB）

================================================================================
一、安全（必看）
================================================================================

1. 电池红线 = 正，黑线 = 负，接反会烧板。
2. 先调 MT3608 空载到 5.0V，再接 ESP32 和 4G。
3. 4G 模块 VIN 必须 5V，禁止接 3.3V。
4. 1000µF 电容：长脚 + 接 5V，短脚 − 接 GND，不能接反。
5. 全部模块 GND 必须连在一起（共地）。
6. 4G 天线插紧；测试时不要用手握天线。
7. 接好线后若发烫、冒烟、有糊味，立即断电。

================================================================================
二、电源系统（TP4056 + 电池 + MT3608 + 电容）
================================================================================

── 二-A：TP4056 与电池 ──────────────────────────────────────

  TP4056 丝印        接什么
  ─────────────────────────────────────────────────────────
  B+  / BAT+         电池红线（+）
  B-  / BAT-         电池黑线（−）
  OUT+               红色导线 → MT3608 IN+
  OUT-               黑色导线 → MT3608 IN-（GND 母线）
  USB Type-C         Type-C 充电线（仅充电，不接 ESP32）

  ★ 建议：先不接电池，用 USB 给 TP4056 充电测试；最后一步再装电池。

── 二-B：MT3608 升压（必须调到 5.0V）────────────────────────

  MT3608 丝印        接什么
  ─────────────────────────────────────────────────────────
  IN+                TP4056 OUT+
  IN-                TP4056 OUT-
  OUT+               5V 红色母线（用万用表测量后再接负载）
  OUT-               GND 黑色母线

  调压步骤：
    1. 只接 TP4056 → MT3608，不接 ESP32 和 4G
    2. 给 TP4056 接 USB 或电池，输入约 3.7～4.2V
    3. 万用表测 MT3608 的 OUT+ 与 OUT- 之间电压
    4. 用小螺丝刀慢慢拧 MT3608 上的电位器，调到 5.0V（4.9～5.1 均可）
    5. 电压稳定后再往下接

── 二-C：5V 母线分配 ─────────────────────────────────────────

  MT3608 OUT+ (5V) ──┬── ESP32 右侧 5V
                     ├── 4G 模块 VIN
                     └── 1000µF 电容 +（靠近 4G 模块）

  MT3608 OUT- (GND) ─┬── ESP32 GND
                     ├── 4G 模块 GND
                     ├── TP4056 OUT-
                     ├── 屏幕 GND
                     ├── 摄像头 GND
                     └── 1000µF 电容 −

  屏幕、摄像头 3.3V ── ESP32 右侧 3V3（各一根，可并联）

  ★ ESP32 调试时可用 USB 线供电；装电池后整机由 MT3608 供电。
    不要 USB 和 MT3608 5V 同时硬并在一起（二选一或只接一种主供电）。

三、4G 模块 FS-MC615-A7670CX（对照模块边缘丝印）
================================================================================

  模块丝印     接到 ESP32 板子丝印         说明
  ─────────────────────────────────────────────────────────
  VIN          独立 5V≥2A（勿只靠 USB）    必须 5V，电流要够
  GND          GND                         共地
  RX           21（左侧排针「21」）        模块收 ← ESP 发（交叉！）
  TX           2（右侧排针「2」）          模块发 → ESP 收（交叉！）
  PWK          47（左侧排针「47」）        开机控制（固件脉冲）
  PEN          48 或 3V3                   使能，必须常高/常接 3.3V
  NET          悬空                        勿接 GPIO2（会顶 UART RX）
  天线座       FPC 天线                    插紧，金属面朝外

  ★ 左侧「TX」「RX」字样两针 = USB 口，不要接 4G！
  ★ 41=屏幕背光，勿再接 PWK；旧表「TX→47 / PWK→41 / NET→2」已作废。
  ★ 没插 SIM 卡也可以先接线测 AT（客户自己插卡）。

四、ST7789 1.3 寸裸屏 12Pin（3.3V，SPI）
================================================================================

  屏幕引脚名（常见）     接到 ESP32 丝印
  ─────────────────────────────────────────────────────────
  GND                    GND
  VCC / 3.3V             3V3
  SCL / SCLK             36
  SDA / MOSI / DIN       35
  CS                     37
  DC / RS                38
  RES / RST              39
  BL / 背光              40（或先接 3.3V 常亮试背光）

  ★ 裸屏 12Pin 各卖家线序可能不同，必须对照客户提供的商家图纸！
  ★ 用万用表确认 VCC 脚后再上电，防止 VCC 接到信号脚烧屏。
  ★ 建议焊接或热缩管固定，杜邦线容易松。

六、拍照按键（2 位独立按键模块，双大方键板）
================================================================================

  客户配件：天猫「2位独立按键模块」（整板，不是单颗 6×6）。
  常见排针丝印：GND / KEY1 / KEY2（有的还有 VCC，接 3V3）。

  接线（装 3D 壳推荐）：
    KEY1 ──► ESP32 丝印「0」或 BOOT     （扫题）
    KEY2 ──► ESP32 丝印「42」           （测 AI）
    GND  ──► GND
    VCC  ──► 3V3（若模块有此脚）

  裸板联调也可先只按板载 BOOT（=GPIO0，等同 KEY1）。
  3D 外壳：后壳左侧壁双键方窗，模块从内侧嵌入。

十二、常见错误（务必避免）
================================================================================

  ✗ 4G 板叠插在 ESP32 排针上           → 必烧或必乱
  ✗ GND 接到 4G VIN                    → 模块无电
  ✗ 4G TX/RX 接到左侧 TX/RX 字样       → 与 USB 冲突
  ✗ 4G VIN 接 3.3V                     → 模块不工作
  ✗ PEN 未接 3.3V                      → 模块不开机
  ✗ 屏幕 VCC 接 5V                     → 烧屏幕
  ✗ MT3608 超过 5.5V                   → 烧 ESP32/4G
  ✗ 电容正负接反                       → 危险
