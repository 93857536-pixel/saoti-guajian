# 配件尺寸 + 杜邦占位（外壳 v7.5 依据）

> 平面尺寸：微雪/商家标注 + 你实测电源板。  
> **高度必须区分「裸板」和「插杜邦后」**。  
> v7.5：按键为天猫 **2 位独立按键模块**（双键 PCB），非单颗 6×6。

## 1. 配件平面尺寸（长 × 宽）

| 配件 | 平面 (mm) | 备注 |
|------|-----------|------|
| **YD-ESP32-S3 N16R8** | **64 × 28.4** | 两侧长边 2.54 排针；USB 在短边 |
| **微雪 1.3" LCD Module** | **45 × 31** | 可视 23.4×23.4；短边出线 |
| **微雪 OV5640 Camera Board (C)** | **35.7 × 23.9 矩形** | 镜头模组偏一端；**18 线杜邦** |
| **电源模块 + ICR14500** | **64 × 35.8** | 高约 18～20 |
| **FS-MCore A7670** | **≈28 × 26** | 含针约 12；再加杜邦 |
| **FPC/PCB 天线** | 40～50 × 8～15 | **外贴**后壳 |
| **2 位独立按键模块** | PCB 约 **32～40 × 18～22** | 双大方键；壳口袋 **42×24**、键距 15.5、方孔 12；KEY1→GPIO0、KEY2→GPIO42 |

## 2. 杜邦几何与线长

| 项目 | 设计取值 |
|------|----------|
| 插合后相对 PCB 高度 | **12 mm**（常见 10～14） |
| 侧向出线占位 | **18 mm** |
| 前线舱净空 Z | **≥18 mm** |
| 线材 | **母对母 20 cm** |
| 设计路径预算 | **≤190 mm**（弯折余量） |

## 3. 裸高 vs 接线后

| 配件 | 裸高 | 接线后设计占位 |
|------|------|----------------|
| ESP32（开发板栈） | 顶面 8～12 | **按 ≥70mm 后仓净高**（杜邦+杂线） |
| LCD | 3～5 | 左右短边出线；优先左侧 |
| OV5640(C) | 镜头 8～12 | 侧向杜邦 |
| 电源 | 18～20 | 20 + 焊盘 |
| A7670 | ≈12 | + 杜邦侧向 |

## 4. 版本要点

| 项 | v7.4 | v7.5 |
|----|------|------|
| 外接键 | 单颗 6×6 Ø12 | **2 位模块双键窗 + 整板凹座** |
| 外廓 / 后仓 | 108 / 72 | 同 |

## 5. v7.5 摆法

| 项 | mm |
|----|-----|
| 外廓 | **106 × 140 × 108** |
| 前壳 / 后壳 | **34 / 74** |
| 后仓净高 | **≈72** |
| 双键模块 | 左壁中心 Y≈18，Z≈22；口袋 42×24 |
