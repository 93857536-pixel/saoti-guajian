先读 [`COMPONENTS.md`](COMPONENTS.md)。生成脚本内置 `AUDIT PASS`。

- v7.3：后仓净高 ≥70mm；20cm 杜邦线径

- **v7.5**：按 [天猫 2 位独立按键模块](https://detail.tmall.com/item.htm?id=610117189167) 整板开双键窗

## 开孔（装机朝向）

| 位置 | 用途 |
|------|------|
| **后壳左侧壁** 双 12×12 方孔 | **2 位按键模块**键帽（内侧整板凹座约 42×24，键距 15.5） |
| **后壳底边** USB-C | 电源充电 |
| **后壳右侧壁** USB-C | ESP 烧录 |
| **后壳** 腰圆孔 | 板载 BOOT（备用） |
| **后壳背面** 指拨长槽 | 电源拨动开关 |
| **顶边** 小孔 | 天线馈线 |
| **后壳外上** 浅凹 | 天线外贴 |

若你的双键板键距/外形差很多：量一下 PCB 长宽和两键中心距，改 `generate_case.py` 里 `BTN_*` 后重跑脚本。

## 按键接线（摘要）

| 模块 | ESP | 功能 |
|------|-----|------|
| KEY1 | GPIO0 | 扫题 |
| KEY2 | GPIO42 | 测 AI |
| GND | GND | |

屏线尽量走左侧短边（20cm 杜邦更宽裕）。详见证线 `WIRING.md`。
