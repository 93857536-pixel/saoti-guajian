# 配件尺寸 + 杜邦占位（外壳 v7.1 依据）

> 平面尺寸：微雪/商家标注 + 你实测电源板。  
> **高度必须区分「裸板」和「插杜邦后」**。  
> v7 全量审计后已出 **v7.1**（修 USB/BOOT/4G 硬伤）。

## 1. 配件平面尺寸（长 × 宽）

| 配件 | 平面 (mm) | 备注 |
|------|-----------|------|
| **YD-ESP32-S3 N16R8** | **64 × 28.4** | 两侧长边 2.54 排针；USB 在短边；下焊 |
| **微雪 1.3" LCD Module** | **45 × 31** | 可视 23.4×23.4 居中；**短边**一侧 PH2.0、一侧 8 孔排针（杜邦出 ±X） |
| **微雪 OV5640 Camera Board (C)** | **35.7 × 23.9 矩形** | 方形的是镜头模组（偏一端）；对侧短边 2×9 排针；**18 线杜邦** |
| **电源模块 + ICR14500** | **64 × 35.8** | 你实测；高约 18～20 |
| **FS-MCore A7670** | **≈28 × 26** | 含针约 12；再加杜邦 |
| **FPC/PCB 天线** | 40～50 × 8～15 | **外贴**后壳，不进仓 |

## 2. 杜邦几何（2.54mm 常用件）

| 项目 | 设计取值 (mm) |
|------|----------------|
| 母头沿出线长度 | ≈14 |
| 插合后相对 PCB 高度 | **12**（常见 10～14） |
| 侧向出线占位 | **17**（胶壳+弯折） |
| 前线舱净空 Z | **≥18** |
| 竖直接出 | 避免（会再加 10mm+ 厚度） |

## 3. 裸高 vs 接线后

| 配件 | 裸高 | 接线后设计占位 |
|------|------|----------------|
| ESP32 | 顶面 8～12 | 侧向杜邦 17；勿按裸高估厚 |
| LCD | 3～5 | 板 + **左右短边** 8 线（PH2.0 或排针杜邦） |
| OV5640(C) | 镜头模组方块+高 8～12 | 排针在短边 → 杜邦出侧向；镜头孔对准模组端（非板心） |
| 电源 | 18～20 | 20 + 焊盘出线 |
| A7670 | ≈12 | + 杜邦侧向 |

### 线束

| 线束 | 根数 | 说明 |
|------|------|------|
| 摄像→ESP | 16～18 | 最粗，专留通道 |
| LCD→ESP | 8 | VCC/GND/DIN/CLK/CS/DC/RST/BL |
| 4G→ESP | ≈6 | 后仓短接 |
| 电源焊盘 | 2～3 | 5V/GND；近 4G 加电容 |

## 4. v7 审计结论 → v7.1

| 检查项 | v7 | v7.1 |
|--------|----|------|
| 屏/摄重叠 | 通过（缝 16.6） | 保持 |
| 杜邦线舱 Z | 通过 | 保持 |
| 电源 USB-C→底孔 | 距内壁 4.1，偏紧 | **贴底 0.6mm** |
| ESP USB→底孔 | **致命：距内壁 55mm** | **改右侧壁，贴壁 0.4mm** |
| BOOT 孔对准 ESP | **致命：孔在电源区** | **后壳孔在 ESP 板内** |
| 4G vs ESP 杜邦道 | **XY 打架** | **4G 改左上** |

## 5. v7.1 外廓与摆法

| 项 | mm |
|----|-----|
| 外廓 | **106 × 140 × 70** |
| 前壳 / 后壳 | **34 / 36** |

```
前壳: 屏(上) → 缝≥16 → 摄像 → 中央杜邦线舱
后壳: 4G(左上) | ESP(右，USB出右侧壁) 
      缝≥18
      电源(偏左贴底，USB-C出底边)
```

1. 屏/摄朝前；杜邦进前舱线槽。  
2. ESP：**USB 朝右壁**；长边排针杜邦朝上/下/左进线舱；器件面朝后 → BOOT 在后壳。  
3. 电源：USB-C 朝底；开关后壳指拨；5V 焊盘。  
4. 天线：后壳外上浅凹，馈线顶孔进壳接 4G。
