## 摆法与接线长度（20cm 杜邦）

```
前壳（上→下）:
  [ 屏 @ Y≈44 ]   建议从左侧短边出线
  —— ≥16mm ——
  [ 摄像 @ Y≈0 ]

后壳（净高≈72mm）:
  [ 4G 左上 ]  [ ESP 右，USB→右壁 ]
  [ 左壁：6×6 拍照键 ]
  —— ≥18mm ——
  [ 电源贴底，USB-C→底边 ]
```

| 线束 | 估算路径 | 相对 200mm |
|------|----------|------------|
| 屏左→ESP | ≈168 mm | 充足 |
| 屏右→ESP | ≈188 mm | 紧，尽量走左侧 |
| 摄像→ESP | ≈122 mm | 充足 |
| 4G / 电源→ESP | ≈62 / 73 mm | 充足 |

外接键：`GPIO0` ↔ `GND`（对角脚），与板载 BOOT 同脚，可并联。
