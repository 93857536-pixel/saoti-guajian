## 摆法

```
前壳（上→下）:
  [ 屏 45×31 ]         方屏居中；8 线从左右短边出（非下沿）
  —— ≥16mm ——
  [ 摄像 35.7×23.9 矩形 ]
      排针朝左(-X)；方形镜头模组偏右；圆孔对准模组（非板心）
      DVP≈18 线从左侧进线舱

后壳:
  [ 4G 左上 ]          避开 ESP
  [ ESP 右侧 ]         USB→右壁；杜邦→上/下/左
  —— ≥18mm ——
  [ 电源 偏左贴底 ]    USB-C→底边
```

### 接线提醒

- 杜邦**侧向**出线，不要竖直堆高。  
- 屏、摄像、ESP、4G 都按杜邦留空（见 COMPONENTS）。  
- 电源 5V/GND 用**焊盘**；近 4G 加滤波电容。
